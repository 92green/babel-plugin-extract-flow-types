# babel-plugin-extract-flow-types (bpeft)


## Installation

### Babel V7

```
yarn add --dev babel-plugin-extract-flow-types
```

### Babel V6

```
yarn add --dev babel-plugin-extract-flow-types@1.0.0
```


## Configuration

```json
{
    "plugins": [
        ["babel-plugin-extract-flow-types", {"srcDir": "src", "outDir": "lib"}]
    ]
}
```
